# EMA Trend Crossover (FX)

EMA Trend Crossover Strategy applied to FX markets (EURUSD, GBPUSD),
designed for trend-following regimes with strict risk control.

## Markets
- EURUSD
- GBPUSD

## Strategy Overview
This strategy uses exponential moving average crossovers to identify
trend-following opportunities in FX markets.

## Current Status
- Backtested on historical data (EURUSD, GBPUSD)
- Forward testing in progress on EURUSD
